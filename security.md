# Security

## Intended Users

The intended users of this repository are:

- **Me**, as the student developing and submitting the homework
- **My professor**, as the reviewer and grader of the submission

This repository is primarily used for coursework submission and evaluation.

## Risk Assessment

The security risk for this repository is relatively low, since it is a homework submission repo and is not intended to store sensitive personal, financial, or production business data. However, there are still some potential concerns if the code or data fell into the wrong hands:

- Someone could copy or reuse my work without permission
- Someone could view assignment solutions or project structure before they are meant to be shared
- If secrets, tokens, or credentials were accidentally committed, they could be exposed
- Unauthorized changes could affect the integrity of the submission

Because this repository is hosted on a regular GitHub account and not UConn Enterprise GitHub, access control and repository protection depend on the GitHub settings configured by the owner (me).

## Steps Taken to Secure the Repository

The following steps have been taken to help secure this repository:

- A **CODEOWNERS** file is included so that repository ownership and review responsibility are clearly defined
- The professor’s GitHub account is listed in the CODEOWNERS file
- The repository is used only for classwork and is not intended to contain sensitive data

At this time, additional security controls such as advanced organization rulesets or enterprise protections are not necessary because this repository is only used for homework submission between the student and professor.
